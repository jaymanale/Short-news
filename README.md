## Run Locally 🖥️

Clone the project

```bash
  git clone https://github.com/jaymanale/Short-news.git
```

Go to the project directory

```bash
  cd Short-news
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
